# ChainSense Privacy Policy

Effective date: February 5, 2026

ChainSense uses this application only to publish and schedule educational content on its own Pinterest business account and to view basic account analytics.

## Data we access
- Pinterest account identifiers and profile metadata
- Pin content and media published by ChainSense
- Basic performance metrics (if enabled)

## How we use data
- Content publishing and scheduling
- Internal reporting and performance analysis

## Data sharing
- We do not sell personal data.
- This app is not offered to public users.
- Access is limited to authorised ChainSense operators.

## Data retention
We retain operational logs only as needed for publishing reliability and compliance.
